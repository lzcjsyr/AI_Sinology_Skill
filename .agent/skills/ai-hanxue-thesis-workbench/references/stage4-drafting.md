# 阶段四：初稿写作

## 输入

- `3_outline_matrix.yaml`
- `2_final_corpus.yaml`

## 输出

- `outputs/<project>/4_first_draft.md`

## 目标

- 根据论纲写出可继续修订的学术初稿。
- 把分析和史料证据绑在一起，不写无来源论断。

## 执行规则

- 按 `sub_sections` 顺序逐段写作。
- 每段只围绕当前小节的 `sub_section_argument` 展开。
- 需要引用原文时，直接取自 `2_final_corpus.yaml` 对应记录，不要改写成不存在的引文。
- 每个小节结尾保留证据锚点行：
  - `证据锚点：\`piece_id_1\`, \`piece_id_2\``

## 写作原则

- 先解释史料语境，再提出分析，不要只堆引文。
- 让每段都能回答“这一证据如何推进论证”。
- 对含混证据要写出限定语，不要过度断言。

## 质量检查

- Markdown 结构与论纲层级一致。
- 所有锚点都能在 corpus 中找到。
- 没有出现脱离证据支撑的大段泛论。
