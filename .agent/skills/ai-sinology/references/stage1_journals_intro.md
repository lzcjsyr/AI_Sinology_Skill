# 阶段一：期刊介绍与选刊路由

适用场景：

- 用户在阶段一需要选刊，或已经点名某家期刊，希望 Skill 反向校准题目、结构与创新表述。
- 题目主要属于中国古代文学、古典文献、古代文论、文学批评史、文艺理论或文献学。

使用规则：

- 本文件是 `references/stage1-planning.md` 的期刊附录，不替代阶段一主工作流。
- 先看“期刊清单”“默认分流”和“跨方向边界”，只做初筛。
- 当用户已经指定期刊，或初筛后锁定 `1-3` 个候选期刊时，再直接跳读本文件中对应期刊的小节，不必从头通读全部内容。
- 如果题目跨两个方向，例如“古代文学 + 文献整理”或“古代文论 + 文艺理论”，最多同时精读 `2` 个期刊小节。

## 期刊清单

### 中国古代文学 / 文献 / 文艺理论方向

- `文学遗产`
  - 适合：古代文学、作家、文体、文学史、小说戏曲、文学批评史问题
  - 详见：本文 `## 文学遗产`
- `文学评论`
  - 适合：文学理论、文学批评、文学史综合问题；古代方向可投，但必须写成文学研究问题
  - 详见：本文 `## 文学评论`
- `文艺理论研究`
  - 适合：文艺理论、古代文论、文学理论、批评理论、理论化程度较高的古代文学研究
  - 详见：本文 `## 文艺理论研究`
- `文艺研究`
  - 适合：美学、文艺学、艺术理论、古代文学与古代文论交叉研究
  - 详见：本文 `## 文艺研究`
- `文献`
  - 适合：文献学理论、版本、校勘、目录、书籍史、敦煌文献、历史文献
  - 详见：本文 `## 文献`
- `文史`
  - 适合：古代文学与文史考证交叉、经典文献辨析、版本源流、语言文字与文学史互证的问题
  - 详见：本文 `## 文史`
- `中国典籍与文化`
  - 适合：典籍、古典文献、传统文化传播、文史交叉、学术考辨
  - 详见：本文 `## 中国典籍与文化`
- `古籍整理研究学刊`
  - 适合：版本、校勘、辑佚、目录、注释是论文骨架，古代文学研究建立在扎实文献整理基础上
  - 详见：本文 `## 古籍整理研究学刊`

## 默认分流

- 核心是古代文学史、作家、文体、小说戏曲：
  - 首看 `文学遗产`
- 核心是文学理论、文学批评、文学史综合问题：
  - 首看 `文学评论`
- 核心是古代文论、诗学、美学范畴：
  - 首看 `文艺理论研究`
  - 次看 `文艺研究`
- 核心是版本、校勘、目录、书籍史、文献整理：
  - 首看 `古籍整理研究学刊`
  - 次看 `文献`
  - 再看 `文史`
  - 再看 `中国典籍与文化`
- 核心是典籍阐释、接受史、传播史、学术史：
  - 首看 `中国典籍与文化`
  - 次看 `文献`
  - 再看 `文史`

## 跨方向边界

- 若论文主要回答“某文本如何被解释、某文学观念如何形成、某作家或文体如何被重估”，优先走中国古代文学或文艺理论期刊。
- 若论文的硬核工作在版本、异文、辑佚、书籍流传，则优先考虑 `文献`、`文史`、`中国典籍与文化` 或 `古籍整理研究学刊`。
- 若论文既有文献整理又有文学解释，阶段一应先判断主问题落在“整理”还是“阐释”，再决定首投刊物。

## 文学遗产

定位摘要：

- `文学遗产` 是中国古代文学研究最硬的一线目标刊物之一。
- 它更关注古代文学、作家、文体、文学史、小说戏曲与文学批评史问题，而不是纯语言学问题。

可确认信息：

- 官方站点显示主办单位为中国社会科学院文学研究所，并提供正式投稿系统与编辑部联系方式。
- 万方期刊导航对该刊的介绍是：发表中国古代文学、古代作家、文学风格和文学流派的研究、评论和考证文章，报道学术界最新研究成果，沟通学术信息。

写作路由：

- 如果题目核心是古代文学问题，而古代汉语分析只是服务于文学解释，这本通常优先于语言学刊物。
- 更适合：
  - 古代作家与作品重估
  - 文体演变、文学流派、批评史
  - 小说、戏曲、诗词、赋学等专题研究
  - 文献基础扎实的文学史问题

不宜写法：

- 只有词义、语法、音韵分析，缺少文学问题意识。
- 只有文献考据，没有转化成文学史或文学批评问题。
- 把材料札记硬写成文学顶刊论文。

阶段一校准建议：

- 引言要先提出文学史或文学理论问题，再说明语言材料如何支撑这个问题。
- 章节结构应围绕文本、作家、文体、接受史或批评史展开，而不是按语言点分章。
- 如果论文主要是版本、异文、辑佚，请优先比较 `文献` 或 `古籍整理研究学刊`。

进一步查询：

- [文学遗产官方投稿系统](https://wxyc.ajcass.com/Home/Index)
- [万方期刊导航页](https://sns.wanfangdata.com.cn/sns/perio/wxyc)

## 文学评论

定位摘要：

- `文学评论` 是文学理论与文学批评领域的顶刊之一。
- 古代方向可以投，但必须把题目写成文学研究问题，而不是古汉语或文献学技术问题。

可确认信息：

- 官方投稿系统页面显示：`文学评论` 为双月刊，中国社会科学院主管，中国社会科学院文学研究所主办，出版单位为社会科学文献出版社。
- 官方页面同时给出编辑部联系方式和正式投稿入口。
- 中国社会科学网相关回顾文章提到：该刊 `1957` 年创刊时名为《文学研究》，`1959` 年改名《文学评论》。

写作路由：

- 适合：
  - 文学理论、文学批评、文学史综合问题
  - 古代文学与现代批评方法的对话
  - 能在较大文学史与理论框架下提出问题的论文

不宜写法：

- 只是古书中若干词语、句法、版本问题，没有文学研究提升。
- 只有作家作品介绍，没有强学术史定位。
- 只做技术性考据，不进入批评史、理论史或文学史问题。

阶段一校准建议：

- 题目宜写成“文学史/批评史/理论问题 + 文本或材料切口”。
- 不能只回答“这个词是什么意思”，要回答“这如何改变我们对某作家、文体、观念或文学史的理解”。
- 如果论文核心仍是古代文学研究而非理论批评，优先比较 `文学遗产`。

进一步查询：

- [文学评论官方投稿系统](https://wxpl.ajcass.com/Home/Index)
- [中国社会科学网相关文章](https://www.cssn.cn/skgz/bwyc/202303/t20230317_5608208.shtml)

## 文艺理论研究

定位摘要：

- `文艺理论研究` 是文艺理论、文学理论、美学研究的重要顶刊。
- 对古代方向而言，它更适合古代文论、诗学、美学概念史、理论化程度高的古代文学研究。

可确认信息：

- 华东师范大学期刊站显示：该刊官网设有“期刊介绍”“投稿指南”“道德声明”等栏目。
- 同站近期期次目录可见，该刊长期设置如“中国自主文论话语体系研究”“古代小说研究”“古代文论与古代文学的理论研究”等专题板块。

写作路由：

- 适合：
  - 古代文论、诗学范畴、理论史
  - 理论化重释经典文献
  - 把古代文学问题提升为文学理论或文艺学问题

不宜写法：

- 纯文学史资料整理，没有理论意识。
- 只有思想史描述，没有文艺理论切口。
- 纯语言学问题。

阶段一校准建议：

- 题目应显出概念、范畴、理论路径，而不只是材料对象。
- 学术史部分要重点梳理不同理论解释路径，而不是只列作品研究。
- 如果你主要想讨论“概念如何生成”“理论如何转型”“古典资源如何进入现代文论”，这本比一般文学史刊物更匹配。

进一步查询：

- [文艺理论研究官网](https://tsla.ecnu.edu.cn/CN/home)
- [文艺理论研究栏目页](https://tsla.ecnu.edu.cn/CN/column/column10.shtml)

## 文艺研究

定位摘要：

- `文艺研究` 是艺术学与文艺学领域的顶级综合期刊。
- 它能接古代文学、古代文论、艺术思想史文章，但前提是题目必须写成文艺学或艺术理论问题，而不是语言学小题。

可确认信息：

- 文化和旅游部官方页面写明：《文艺研究》创刊于 `1979` 年，是文化和旅游部主管、中国艺术研究院主办的大型综合性文艺理论期刊。
- 同页写明：该刊被评为艺术学顶级期刊，刊登论文涵盖文学艺术各门类，包括一般理论、美学文艺学、文学戏剧影视造型艺术等理论与实践研究，以及外国文艺理论、文艺思潮、文艺流派研究。
- 同页还给出了分学科投稿邮箱，其中 `wyyj02@vip.163.com` 对应中国古代文学、中国现当代文学、外国文学。

写作路由：

- 适合：
  - 古代文论、美学范畴、诗学、批评史
  - 古代文学与艺术理论交叉
  - 理论化程度较高的古代文学研究
  - 能与文艺学、艺术学公共议题对话的文章

不宜写法：

- 只有文本考释，没有理论提升。
- 纯版本校勘、纯训诂、纯语法分析。
- 题目太微观，无法进入文艺理论或艺术学层面。

阶段一校准建议：

- 引言必须先回答“这是一个什么文艺理论问题”，再谈材料。
- 需要更强的方法意识、理论语言和问题抽象能力。
- 如果古代文学只是例证，而真正目标是美学、诗学、艺术理论，这本更合适。

进一步查询：

- [文化和旅游部官方期刊介绍](https://www.mct.gov.cn/gywhb/jgsz/zgxwcbdw_jgsz/zzs_zgxwcbdw/202507/t20250722_961326.htm)

## 文献

定位摘要：

- `文献` 是古典文献学、版本目录学、书籍史、敦煌文献与历史文献研究的核心期刊之一。
- 它适合“文献整理是论文骨架”的题目，不适合纯文学评论或纯语言学理论稿。

可确认信息：

- 国家图书馆“中国古籍保护网”页面明确写明：《文献》是文化和旅游部主管、国家图书馆主办的国家级学术刊物。
- 同页写明：该刊既重视传统文献学理论与方法，也设置书籍史研究、史源学与历史书写、编纂与出版等专栏；并连续多年获 AMI 核心、CSSCI 来源期刊等评价。

写作路由：

- 适合：
  - 版本、校勘、目录、辨伪
  - 书籍史、编纂史、知识传播史
  - 敦煌文献、石刻文献、档案文献、历史文献
  - 依托馆藏材料、稿本、抄本、刻本展开的研究

不宜写法：

- 只做文学阐释，没有文献工作量。
- 只做语法、词义、音韵分析，但没有版本或文献整理层面的推进。
- 抽象理论很多，具体文献抓手很弱。

阶段一校准建议：

- proposal 中必须明确使用哪一类文献、哪种版本、何种整理路径。
- 题目最好显出“文献对象”与“学术动作”，如校勘、考源、编纂、流传、抄刻、目录、书籍史。
- 如果你的语言问题建立在馆藏材料或异文系统之上，这本往往比一般文学刊物更合适。

进一步查询：

- [国家图书馆《中国古籍保护网》文献栏目](https://www.nlc.cn/pcab/zy/wx/index.shtml)
- [《文献》2024年第6期目录与摘要](https://www.nlc.cn/pcab/zx/xw/20241118_2642018.shtml)

## 文史

定位摘要：

- `文史` 是文史哲综合研究中的老牌重要刊物，适合古代文学与文献考证、语言文字、历史材料互证相结合的论文。
- 它比 `文学遗产` 更强调考据、辨析和材料根基，比 `文献` 更能容纳文学史问题，但整体仍偏重扎实的文史考证。

可确认信息：

- 维普期刊页将 `文史` 概括为发表中国历史、文学、哲学、语言文字等方面研究考证文章的重要刊物。
- 现有公开介绍与回顾材料可见，该刊长期与中华书局的古典学术出版传统密切相关，基本路数稳定在文史考证与经典研究。

写作路由：

- 适合：
  - 古代文学与文史考证交叉
  - 经典文献辨析、版本源流、文本形成
  - 作家、作品、文体问题中的史料互证
  - 语言文字、训释材料能够实质推进文学史判断的论文

不宜写法：

- 只有纯文学阐释，缺少材料辨析与考证支撑。
- 只有纯技术性版本校勘，没有进一步形成可讨论的文史问题。
- 题目过于宏观，缺少可落地的文本与史料抓手。

阶段一校准建议：

- 题目最好同时显出研究对象与考证动作，不要只写成泛泛的文学史判断。
- 引言要先说明本文要解决的文史争点，再交代材料来源与辨析路径。
- 如果论文核心是纯古代文学阐释，优先比较 `文学遗产`；如果核心是纯文献整理，优先比较 `文献` 或 `古籍整理研究学刊`。

进一步查询：

- [维普《文史》期刊页](https://www.cqvip.com/journal/821159/973800)
- [澎湃关于《文史》与中华书局的回顾材料](https://www.thepaper.cn/newsDetail_forward_18094490)

## 中国典籍与文化

定位摘要：

- `中国典籍与文化` 是典籍、古典文献、传统文化研究的重要交叉刊物。
- 它比 `文献` 更开放于文史哲交叉和文化传播议题，但仍要求扎实的文献基础。

可确认信息：

- 北京大学中文系页面写明，本刊唯一投稿渠道为 `ccc@pku.edu.cn`。
- 同页写明：来稿篇幅以中文 `1.5` 万字内为宜，较长论文与珍稀文献发布可改投《中国典籍与文化论丛》。
- 全国高等院校古籍整理研究工作委员会页面介绍，该刊主要面向知识群体，阐述文化理念，发布学术新见，交流治学心得，沟通资料信息。

写作路由：

- 适合：
  - 典籍阐释与文化意义重估
  - 古典文献与学术史、接受史、传播史问题
  - 文史哲交叉、传统文化专题、海外汉籍与国际汉学
  - 兼具材料基础与文化解释能力的论文

不宜写法：

- 过于技术性的校勘细节，没有更大问题意识。
- 纯理论空转，没有典籍依据。
- 纯语言学稿件，尤其是只讨论局部词法句法。

阶段一校准建议：

- 题目应同时体现“典籍对象”和“文化/学术问题”，不要只写成“某书校记数则”。
- 可以把材料问题提升到文化传播、经典建构、接受史或学术史层面。
- 如果论文太长或更偏资料发布，应考虑其论丛体系。

进一步查询：

- [北京大学中文系期刊页](https://chinese.pku.edu.cn/xsyj/xsqk/da6dfa47c7b543018fa83637753a54e7.htm)
- [全国高校古委会《中国典籍与文化》最新一期](https://gwh.pku.edu.cn/zbkw/zgdjywh_zxyq.htm)
- [“中国典籍与文化”系列讲座介绍](https://gwh.pku.edu.cn/info/1158/1348.htm)

## 古籍整理研究学刊

官方可确认信息：

- 东北师范大学文学院页面写明，《古籍整理研究学刊》为双月刊，`1985` 年创刊。
- 官方页面写明，该刊由教育部全国高等院校古籍整理研究工作委员会主管，东北师范大学文学院古籍整理研究所主办，是国内唯一专门刊载古籍整理研究成果的刊物。
- 官方页面列出的栏目包括：文献研究、文献考订、版本研究、文献目录、校勘注释、古籍辑佚、语言文献、语言文字、出土文献、文学与文化、历史与文化等。

写作路由：

- 如果工作核心是版本、目录、校勘、辑佚、注释，这本通常比纯语言刊物更合适。
- 对古代汉语研究者而言，最适合的情形是：语言问题必须建立在严密的文献整理基础上。
- 若文章主体是“校本重建 + 异文比勘 + 由此引出语言判断”，这里很合适。

适配题型：

- 某书版本系统与语言解释关系
- 校勘、注释、辑佚对词义、语法、语音判断的修正
- 出土文献与传世文献互证下的文献语言问题

不宜写法：

- 脱离文献基础的纯理论语言学论文
- 只有语言分析，没有版本或文献整理支撑
- 只做泛泛文化阐释，不见具体文献工夫

阶段一校准建议：

- proposal 中必须写清底本文本、版本系统和整理方法。
- 章节逻辑往往要先整理文献，再进入语言或思想分析。
- 阶段二不仅要检索正文片段，还要记录版本、异文、校记来源。
