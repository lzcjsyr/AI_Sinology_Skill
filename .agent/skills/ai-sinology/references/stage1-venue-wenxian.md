# 文献

定位摘要：

- `文献` 是古典文献学、版本目录学、书籍史、敦煌文献与历史文献研究的核心期刊之一。
- 它适合“文献整理是论文骨架”的题目，不适合纯文学评论或纯语言学理论稿。

可确认信息：

- 国家图书馆“中国古籍保护网”页面明确写明：《文献》是文化和旅游部主管、国家图书馆主办的国家级学术刊物。
- 同页写明：该刊既重视传统文献学理论与方法，也设置书籍史研究、史源学与历史书写、编纂与出版等专栏；并连续多年获 AMI 核心、CSSCI 来源期刊等评价。

写作路由：

- 适合：
  - 版本、校勘、目录、辨伪
  - 书籍史、编纂史、知识传播史
  - 敦煌文献、石刻文献、档案文献、历史文献
  - 依托馆藏材料、稿本、抄本、刻本展开的研究

不宜写法：

- 只做文学阐释，没有文献工作量。
- 只做语法、词义、音韵分析，但没有版本或文献整理层面的推进。
- 抽象理论很多，具体文献抓手很弱。

阶段一校准建议：

- proposal 中必须明确使用哪一类文献、哪种版本、何种整理路径。
- 题目最好显出“文献对象”与“学术动作”，如校勘、考源、编纂、流传、抄刻、目录、书籍史。
- 如果你的语言问题建立在馆藏材料或异文系统之上，这本往往比一般文学刊物更合适。
