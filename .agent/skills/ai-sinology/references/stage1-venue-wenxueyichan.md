# 文学遗产

定位摘要：

- `文学遗产` 是中国古代文学研究最硬的一线目标刊物之一。
- 它更关注古代文学、作家、文体、文学史、小说戏曲与文学批评史问题，而不是纯语言学问题。

可确认信息：

- 官方站点显示主办单位为中国社会科学院文学研究所，并提供正式投稿系统与编辑部联系方式。
- 万方期刊导航对该刊的介绍是：发表中国古代文学、古代作家、文学风格和文学流派的研究、评论和考证文章，报道学术界最新研究成果，沟通学术信息。

写作路由：

- 如果题目核心是古代文学问题，而古代汉语分析只是服务于文学解释，这本通常优先于语言学刊物。
- 更适合：
  - 古代作家与作品重估
  - 文体演变、文学流派、批评史
  - 小说、戏曲、诗词、赋学等专题研究
  - 文献基础扎实的文学史问题

不宜写法：

- 只有词义、语法、音韵分析，缺少文学问题意识。
- 只有文献考据，没有转化成文学史或文学批评问题。
- 把材料札记硬写成文学顶刊论文。

阶段一校准建议：

- 引言要先提出文学史或文学理论问题，再说明语言材料如何支撑这个问题。
- 章节结构应围绕文本、作家、文体、接受史或批评史展开，而不是按语言点分章。
- 如果论文主要是版本、异文、辑佚，请优先比较 `文献` 或 `古籍整理研究学刊`。
