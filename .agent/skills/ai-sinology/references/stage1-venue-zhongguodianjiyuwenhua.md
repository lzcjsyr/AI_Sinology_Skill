# 中国典籍与文化

定位摘要：

- `中国典籍与文化` 是典籍、古典文献、传统文化研究的重要交叉刊物。
- 它比 `文献` 更开放于文史哲交叉和文化传播议题，但仍要求扎实的文献基础。

可确认信息：

- 北京大学中文系页面写明，本刊唯一投稿渠道为 `ccc@pku.edu.cn`。
- 同页写明：来稿篇幅以中文 `1.5` 万字内为宜，较长论文与珍稀文献发布可改投《中国典籍与文化论丛》。
- 全国高等院校古籍整理研究工作委员会页面介绍，该刊主要面向知识群体，阐述文化理念，发布学术新见，交流治学心得，沟通资料信息。

写作路由：

- 适合：
  - 典籍阐释与文化意义重估
  - 古典文献与学术史、接受史、传播史问题
  - 文史哲交叉、传统文化专题、海外汉籍与国际汉学
  - 兼具材料基础与文化解释能力的论文

不宜写法：

- 过于技术性的校勘细节，没有更大问题意识。
- 纯理论空转，没有典籍依据。
- 纯语言学稿件，尤其是只讨论局部词法句法。

阶段一校准建议：

- 题目应同时体现“典籍对象”和“文化/学术问题”，不要只写成“某书校记数则”。
- 可以把材料问题提升到文化传播、经典建构、接受史或学术史层面。
- 如果论文太长或更偏资料发布，应考虑其论丛体系。
