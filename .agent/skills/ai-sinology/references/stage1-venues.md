# 阶段一：目标期刊清单

适用场景：

- 用户在阶段一需要选刊，或已经点名某家期刊，希望 Skill 反向校准题目、结构与创新表述。
- 题目主要属于中国古代文学、古典文献、古代文论、文学批评史、文艺理论或文献学。

使用规则：

- 本文件是 `references/stage1-planning.md` 的选刊附录，不替代阶段一主工作流。
- 先读本文件，只做初筛，不要一口气加载所有单刊 reference。
- 当用户已经指定期刊，或初筛后锁定 `1-3` 个候选期刊时，再只读取对应的单刊 reference。
- 如果题目跨两个方向，例如“古代文学 + 文献整理”或“古代文论 + 文艺理论”，最多同时再读 `2` 个单刊 reference。

## 期刊清单

### 中国古代文学 / 文献 / 文艺理论方向

- `文学遗产`
  - 适合：古代文学、作家、文体、文学史、小说戏曲、文学批评史问题
  - 再读：`references/stage1-venue-wenxueyichan.md`
- `文学评论`
  - 适合：文学理论、文学批评、文学史综合问题；古代方向可投，但必须写成文学研究问题
  - 再读：`references/stage1-venue-wenxuepinglun.md`
- `文艺理论研究`
  - 适合：文艺理论、古代文论、文学理论、批评理论、理论化程度较高的古代文学研究
  - 再读：`references/stage1-venue-wenyililunyanjiu.md`
- `文艺研究`
  - 适合：美学、文艺学、艺术理论、古代文学与古代文论交叉研究
  - 再读：`references/stage1-venue-wenyiyanjiu.md`
- `文献`
  - 适合：文献学理论、版本、校勘、目录、书籍史、敦煌文献、历史文献
  - 再读：`references/stage1-venue-wenxian.md`
- `文史`
  - 适合：古代文学与文史考证交叉、经典文献辨析、版本源流、语言文字与文学史互证的问题
  - 再读：`references/stage1-venue-wenshi.md`
- `中国典籍与文化`
  - 适合：典籍、古典文献、传统文化传播、文史交叉、学术考辨
  - 再读：`references/stage1-venue-zhongguodianjiyuwenhua.md`
- `古籍整理研究学刊`
  - 适合：版本、校勘、辑佚、目录、注释是论文骨架，古代文学研究建立在扎实文献整理基础上
  - 再读：`references/stage1-venue-gujizhengliyanjiuxuekan.md`

## 默认分流

- 核心是古代文学史、作家、文体、小说戏曲：
  - 首看 `文学遗产`
- 核心是文学理论、文学批评、文学史综合问题：
  - 首看 `文学评论`
- 核心是古代文论、诗学、美学范畴：
  - 首看 `文艺理论研究`
  - 次看 `文艺研究`
- 核心是版本、校勘、目录、书籍史、文献整理：
  - 首看 `古籍整理研究学刊`
  - 次看 `文献`
  - 再看 `文史`
  - 再看 `中国典籍与文化`
- 核心是典籍阐释、接受史、传播史、学术史：
  - 首看 `中国典籍与文化`
  - 次看 `文献`
  - 再看 `文史`

## 跨方向边界

- 若论文主要回答“某文本如何被解释、某文学观念如何形成、某作家或文体如何被重估”，优先走中国古代文学或文艺理论期刊。
- 若论文的硬核工作在版本、异文、辑佚、书籍流传，则优先考虑 `文献`、`文史`、`中国典籍与文化` 或 `古籍整理研究学刊`。
- 若论文既有文献整理又有文学解释，阶段一应先判断主问题落在“整理”还是“阐释”，再决定首投刊物。

## 加载上限

- 阶段一默认只读：
  - `references/stage1-planning.md`
  - `references/stage1-venues.md`
- 若要深度校准，再追加：
  - `1` 个目标刊物单刊 reference
  - 必要时再加 `references/a-journal-writing.md`
- 除非用户明确要求比较多家刊物，否则不要一次读取超过 `2` 个单刊 reference。
